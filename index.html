Objetivo

Desenvolver uma lixeira automática que abre ao detectar aproximação, utilizando Arduino, sensor de distância, servo motor e buzzer. Também é objetivo criar um site em HTML e CSS que apresente o projeto de forma clara, didática e acessível.
